INTRODUCTION TO CSS

CSS also known as cascading style sheet is an aspect of web development that is used to beautify a website and make it look presentable

WAYS TO WRITE CSS

  1. IN-LINE CSS
  2. INTERNAL CSS
  3. EXTERNAL CSS

IN-LINE CSS:In-line css is an a way of styling your website by coding in sideof the element tag using the "style" attribute.

INTERNAL CSS:Internal css is a means of styling the website using the "style" attribute but unlike in-line css where you style in the body in internal css you style in the head using style tag and adding a class to every single element

EXTERNAL CSS:This is the best way of styling where-by you create a new file and named "---.css".This method makes it easier to identify thr element behind styled
